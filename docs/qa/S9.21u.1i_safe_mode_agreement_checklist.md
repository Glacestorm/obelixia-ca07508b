SafeMode logic, base de datos, RLS, edge functions, migraciones ni seeds.

- [ ] Seleccionar **Carlos Ruiz Martín**.
- [ ] Verificar que el bloque ámbar de SafeMode aparece en la parte superior.

  - [ ] No aparece truncado tipo "agreement.extra_pay…".
